# Nyebluck
Nyebluck adalah aplikasi kasir berbasis mobile yang dirancang untuk membantu pelaku usaha dalam mengelola transaksi penjualan, data produk (topping/menu), serta manajemen staff secara praktis. Aplikasi ini dilengkapi dengan fitur laporan dan pembuatan struk dalam bentuk PDF, serta dibangun menggunakan Flutter dengan backend Supabase.
